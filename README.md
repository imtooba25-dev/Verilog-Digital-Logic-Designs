# Digital Design & Verification Library (Verilog)

A collection of foundational combinational and sequential digital logic modules, developed and verified using behavioral and structural Verilog.

## Contents
* **Combinational Logic:** Half/Full Adders, Ripple Carry Adder (RCA), Subtractors, and Multiplexers (4:1).
* **Sequential Logic:** Counters and D-Flip-Flops with asynchronous reset configurations.
* **Verification:** Dedicated testbenches (`_tb.v`) generating Value Change Dump (`.vcd`) files for waveform simulation analysis.
